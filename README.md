# TPS
ThirdPersonShooter

# Controls
TAB: Scoreboard
WASD: Movement
SPACE: Jump
LEFT CLICK: Fire
RIGHT CLICK: Aim
E: Pickup weapon

# Features
- 5 weapons

- 2 weapon handling modes

- 2 aiming states

- weapon pickups

- hitmarkers

- scoreboard

- match srart countdown

- match timer

- respawn countdown

- ammo counter & weapon info

- health & health bar

- reloading indicator
